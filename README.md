Domutil
===============
A module for manipulating the DOM or control the DOM events.

## Feature
* domutil.js
  * Module that can control the information of DOM
  * Add, remove, find DOM class name
* domevent.js
  * DOM events module
  * Add, remove, fire DOM events
  * Control mouse events
